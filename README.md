Google Custom Search Engine (CSE)
===========

Install Google Custom Search on your forum.

[![Build Status](https://travis-ci.org/ForumHulp/gcse.svg?branch=master)](https://travis-ci.org/ForumHulp/gcse)

## Requirements
* phpBB 3.1.0-dev or higher
* PHP 5.3.3 or higher

## Installation
You can install this extension on the latest copy of the develop branch ([phpBB 3.1-dev](https://github.com/phpbb/phpbb3)) by doing the following:

1. Copy the [entire contents of this repo](https://github.com/ForumHulp/gcse/archive/master.zip) to `FORUM_DIRECTORY/ext/forumhulp/gcse/`.
2. Navigate in the ACP to `Customise -> Extension Management -> Manage extensions`.
3. Click Google Custom Search Engine (CSE) => `Enable`.

## Update
1. Download the [latest ZIP-archive of `master` branch of this repository](https://github.com/ForumHulp/gcse/archive/master.zip).
2. Navigate in the ACP to `Customise -> Extension Management -> Manage extensions` and click oogle Custom Search Engine (CSE) => `Disable`.
3. Copy the contents of `errorpages-master` folder to `FORUM_DIRECTORY/ext/forumhulp/gcse/`.
4. Navigate in the ACP to `Customise -> Extension Management -> Manage extensions` and click Google Custom Search Engine (CSE) => `Enable`.
5. Click `Details` or `Re-Check all versions` link to follow updates.
6. Or use our Upload Extensions extension

## Uninstallation
Navigate in the ACP to `Customise -> Extension Management -> Manage extensions` and click Google Custom Search Engine (CSE) => `Disable`.

To permanently uninstall, click `Delete Data` and then you can safely delete the `/ext/forumhulp/gcse/` folder or use our Upload Extensions extension to delet all files and folders.
